# Screenshot Notes

This folder stores public README screenshots captured with the synthetic `Falcon Trading DMCC, demo only` dataset.

Capture guidance:

- Use a wide viewport, preferably 1440x1000 or 1600x1000.
- Keep browser zoom consistent at 90% or 100%.
- Keep the Streamlit sidebar collapsed.
- Avoid cropped titles or horizontally clipped content.
- Use only synthetic demo data.
- Keep the public README to the strongest 4 to 6 screenshots.
